# 재귀호출을 이용한 문제 해결

* 재귀함수를 디자인 하기 위한 세 가지 단계

1. 함수의 **정의**를 명확히 한다
2. **기저 조건**에서 함수가 제대로 동작하게 작성한다
3. 함수가 제대로 동작한다고 가정하고 **함수를 완성**한다

# 이진탐색

* 시간복잡도 O(log N)

# 분할정복법

* 문제를 소문제로 **분할**
* 각각의 **소문제**를 해결
* 소문제의 해결 결과를 이용해 **전체 문제**를 해결
* 어려움
* **수학적 문제 해결 능력**이 가장 중요
* 키보드 대신에 **노트와 펜**을 들고 생각
